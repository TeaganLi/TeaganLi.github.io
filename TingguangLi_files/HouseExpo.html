HouseExpo Dataset

Autonomously exploring a priori unknown environment is an essential task for mobile robots. Due to the lack of a unified experimental platform, current works towards this problem are often evaluated in a few manually designed scenarios. Meanwhile, with the recent development of deep learning techniques, some researchers attempt to apply learning based methods to the exploration problem, which requires a substantial amount of data. To satisfy the underlying demand, in this paper we build HouseExpo, a large-scale indoor layout dataset, ranging from single-room studios to multi-room houses. In our paper, we conducted three experiments and the training set and the testing set can be downloaded through the following link. And the whole dataset together with the simulator will come soon.

Deep Reinforcement Learning based Exploration

With an emphasis on generalization ability, we use HouseExpo to benchmark deep reinforcement learning based methods on the robot exploration problem. The experiments show that the trained agent indeed learns some useful skills like avoiding obstacles and moving along walls. It is also verified that the generalization performance is affected by the complexity of environments and the training set size. A short explanation video is attached below.
